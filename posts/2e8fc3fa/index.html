寒假马上要结束，预计2.17返校，是时候回顾一下这一个多月，然后规划一下接下来的计划。

提高

假期当然也是要学点东西的，虽然效率并不高，但是还是有一些收获的。

语言（计算机）

C++

看了大半本《C++ Primer Plus》，也做了相关的课后题，但感觉还是晦涩难懂，也仅仅是看了一遍，还需要多看几遍。

Python

Python 其实常用的语法以及很熟悉了，毕竟写了那么多的脚本，但是还是有一些不熟悉的地方，比如说多线程，多进程，还有一些库的使用，比如说numpy，pandas，matplotlib等等。

然后找了一个相关的稍微补充了一下，但是感觉还是不够，还是需要多看几遍。链接在这里：Python

Robot

ROS

ROS2的熟悉程度提高了，但也只是提高了一丢丢，没玩明白。

尝试了一下 ROS2 分布式，还不错。

尝试去获取别人写好的包的信息，也成功了。链接

moveit

尝试了一下moveit，真的吃内存和cpu，笔记本还是不太行哇，勉强玩玩。

其他

吴恩达的机器学习，就看了2节的样子，然后转头去看了李沐的《动手学深度学习》，虽然是中文，但还是不懂。看了好几章但并不理解，感觉还是需要多看几遍。

漫画自动控制课，也是看不下去，可能需要静下心来看一下。

科研（打工）

被拉去帮航天五院的工程师调试设备，体验一般。吸取教训，以后自己的东西不要做成那样。

自己的毕业课题博士去申请国家基金，也不知道能不能成，加油！！！

假期写了几篇软著，其实就是把之前的东西整理了一下，形成文档，加加综测，想拿一等。

生活

日常

日常的生活还是很懒散的，大多数都在摸鱼，但是也有一些收获，我觉得还算及格吧。

假期除了走亲戚和陪我爸妈逛了两圈，其他时间基本都呆在家里，也没怎么出去玩。

春节档的电影找的资源看的，还行。

电视剧 《狂飙》 去看
